Pandabin

Pandabin is a light-weight (fewer than 600 sloc) pastebin written in C.

It was inspired by pb and ompldr (now defunct).

Routes

  • POST /create

    Example: curl -X POST --data-binary @.zshrc 'localhost:8080'

  • GET /read?hash=<hash>&ext=<extension>

    Example: curl 'localhost:8080/read?hash=8e764a7a65715459d87084fa5f30e6f5209c22d42c117a7a399dc1debe6b1388'

  • DELETE /delete?uuid=<UUID>

    Example: curl -X DELETE 'localhost:8080/delete?uuid=01892f4d-995e-4fa5-9fa8-08a6d514937e'
